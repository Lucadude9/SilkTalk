# SilkTalk

Team members
 Lucas Pimentel, Alex Elleazar De Guzman, Bhuvnesh Sharma
Team lead
Lucas Pimentel
Project Topic
Pop culture: reviews and discussions
Project Scope
To create a website in which users are able to discuss with others online the following topics: video games, comic, novels, music, movies, and tv shows. We want to create a safe space where the sole purpose is for people to find others with similar interests to create discussions and review media within pop culture. We would like for our website to have different categories based on what the user would like to discuss and get off their mind on their trip to our website
